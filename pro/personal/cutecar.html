Egregore

A first-person adventure into a mysterious café in Cairo.

Introduction

Egregore is a first-person adventure set in a mysterious café in Cairo. Players adopt a new perspective and shape reality to unravel an occult conspiracy hidden in plain sight. The gameplay involves point-and-click mechanics, gathering clues, solving puzzles, and connecting graph nodes.

My Contributions

Character Modeling

I was responsible for creating low-poly character models that emphasized silhouettes for style and performance efficiency.

Pipeline and Workflow

This was a project with a large team, so I wanted to create a cohesive rigging pipeline. The rig and models are made using Maya, then implemented into Unreal Engine.

Unreal Engine is really powerful so we decided to simply rig the characters and then apply the animations in the engine.

Lessons Learned

Working on Egregore challenged me to balance artistic expression with technical efficiency. I learned how to optimize assets for real-time rendering while maintaining visual fidelity, ensuring a smooth transition from concept to in-game implementation.
